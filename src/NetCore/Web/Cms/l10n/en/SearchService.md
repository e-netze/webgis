#service_url: Service URL

The URL for the search service. If the service is a protected LuceneServerNET service, authentication can be attempted via this URL: https://client:secret@server.com

#index_name: Optional: Index name

The index name is already defined in the service URL for most search services. The exception is search services of type 'LuceneServerNET'. Specifying the index name is mandatory for these.
