#max_level: Maximum Level

The highest level that can be used for this service. 
A value less than 0 indicates that the maximum level corresponds to the maximum matrix set level from the Capabilities.

#hide_beyond_max_level: Hide below max level

If the user zooms in further on the map than this tiling service allows, the tiles will no longer be displayed. 
By default (value = false), the service is still displayed, and the tiles are shown accordingly, enlarged/blurred.
