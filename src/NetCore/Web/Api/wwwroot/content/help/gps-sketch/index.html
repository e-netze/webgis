Kurzbeschreibung

Um die GPS Verfolgung zu aktivieren, die Bubble ziehen.

Detailierte Beschreibung

Mit diesem Werkzeug können Stützpunkte für den Sketch aus der GPS Position übernommen werden. Dazu zieht man die "Bubble" in die Karte:

Wenn die GPS Verfolgung aktiv ist, kann man die normalen Zeichen-Werkzeuge nicht mehr verwenden. Ein Stützpunkt kann nur mehr aus dem GPS übernommen werden. Dies erfolgt, wenn man auf die aktive GPS-"Bubble" klickt. Bei jedem Klick wird ein neuner Stützpunkt gezeichnet. Die GPS Verfolgung kann wieder beendet werden, wenn die "Bubble" wieder zurück in den grauen Bereich geschoben wird. Danach kann man die Zeichenwerkzeuge wieder in gewohnter Weise benutzen.

Nach dem Starten der GPS Verfolgung, erscheint die "Bubble" zuerst rot. In diesen Zustand, kann man mit der "Bubble" keine Stützpunkte erfassen. Der Grund ist, dass hier das GPS erst Initialisiert werden muss oder die gewünschte Genauigkeit noch nicht erreicht ist.

Ist das GPS Initialisiert und die gewünschte Genauigkeit erreicht, erscheint die "Bubble" in grün. Ab jetzt können durch einen Klick auf die "Bubble" Stützpunkt erfasst werden.

Drückt man die "Bubble" länger als eine Sekunde, ohne sie dabei zu verschieben, öffnet sich beim loslassen ein Fenster mit Informationen über die mindestens geforderte Genauigkeit und falls verhanden die verwendetet Helmerttransformation.
